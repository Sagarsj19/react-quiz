# React Quiz

A quiz application built with React that allows users to answer questions, track progress, and view their final score.

## Features

- Fetches questions from JSON Server
- Tracks quiz progress
- Displays remaining questions
- Calculates score
- Built with React Hooks
- Responsive UI

## Technologies

- React
- JavaScript (ES6+)
- JSON Server
- CSS

## Installation

npm install

## Run React App

npm start

## Run JSON Server

npm run server
